# INT 499 StreamList Capstone Project

This repository contains the StreamList capstone project for INT 499 at the University of Arizona Global Campus.

## Project Overview
StreamList is a web-based application developed for EZTechMovie that allows users to create and manage personalized streaming lists. The project is being developed incrementally over five weeks and includes planning, design, development, and documentation artifacts.

## Contents
- React StreamList application source code
- Software Requirements Specification (SRS)
- Gantt chart and project timeline
- Wireframes and navigation flow diagrams
- Project status reports
